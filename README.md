# template-git-setup

Quick setup — if you've done this kind of thing before  
or clone directly:  

```bash
git clone git@github.com:dragonsecurity-incubating/template-git-setup.git
```

Get started by creating a new file or uploading an existing one.  
We recommend every repository include a README, LICENSE, and .gitignore.

…or create a new repository on the command line

```bash
echo "# template-git-setup" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin git@github.com:dragonsecurity-incubating/template-git-setup.git
git push -u origin main
```

…or push an existing repository from the command line

```bash
git remote add origin git@github.com:dragonsecurity-incubating/template-git-setup.git
git branch -M main
git push -u origin main
```
